# Nazik's Timer+

Мобильное PWA-приложение для длинных таймеров, которые можно добавить на экран домой на iPhone.

## Запуск

Открой `index.html` в браузере или запусти локальный сервер:

```bash
python3 -m http.server 4173
```

После этого приложение будет доступно на `http://localhost:4173` или на другом свободном порту.
